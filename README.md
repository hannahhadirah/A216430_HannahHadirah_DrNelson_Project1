# FreshKeeper - Project 1

## Project Description
FreshKeeper (Project 1) is a mobile application developed to help users manage food items and reduce food waste by tracking expiry dates. The app allows users to store food information, view it in a structured list and access basic recipe suggestions.

This project focuses on learning fundamental Android development concepts using Jetpack Compose, navigation, and basic data handling.

## SDG Theme
SDG 12: Responsible Consumption and Production  
This application helps reduce food waste by encouraging users to track food expiry dates and manage food items more effectively.

## Features
- Add food items with name and expiry date
- View list of stored food items
- View food detail screen
- Simple recipe suggestions
- Navigation between multiple screens
- Summary screen for basic overview

## Tech Stack
- Kotlin
- Jetpack Compose
- Android Studio
- Navigation Component

## App Screens
- Home Screen
- Add Food Screen
- Food List Screen
- Food Detail Screen
- Recipe Screen
- Summary Screen

## Learning Outcome
Through this project, I learned the basics of Jetpack Compose UI development, navigation and local data storage using Room. I also learned how to structure a multi-screen application and manage simple user interactions.


## Limitations
- No API integration
- No Firebase cloud storage
- No sensor integration
- No real-time data fetching

These features were later improved in Project 2.
